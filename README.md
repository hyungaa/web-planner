# 🌐 Web Planner

## ✨ 프로젝트 소개 (Project Introduction)

이 프로젝트는 웹 기반의 간단한 플래너 애플리케이션입니다. 사용자들은 할 일을 추가하고, 오늘의 할 일을 필터링하며, 완료된 할 일을 체크하고, 모든 데이터를 로컬 스토리지에 저장할 수 있습니다. 개인적인 일정 관리나 간단한 할 일 목록을 웹에서 편리하게 관리하고 싶은 분들을 위해 제작되었습니다.

This project is a simple web-based planner application. Users can add to-do items, filter tasks for today, mark tasks as complete, and save all data to local storage. It's designed for individuals who want to conveniently manage their personal schedules or simple to-do lists directly from their web browser.

## 🚀 주요 기능 (Key Features)

*   **할 일 추가 (Add To-Do):** 새로운 할 일을 쉽게 추가할 수 있습니다.
*   **오늘 할 일 필터링 (Filter Today's To-Do):** 오늘 날짜에 해당하는 할 일만 볼 수 있도록 필터링합니다.
*   **할 일 완료 체크 (Complete Check):** 완료된 할 일을 표시하여 진행 상황을 추적할 수 있습니다.
*   **할 일 삭제 (Delete To-Do):** 불필요한 할 일은 언제든지 삭제할 수 있습니다.
*   **로컬 스토리지 저장 (Local Storage Save):** 모든 할 일 데이터가 브라우저의 로컬 스토리지에 자동 저장되어 페이지를 닫았다가 다시 열어도 데이터가 유지됩니다.

## ▶️ 실행 방법 (How to Run)

이 프로젝트는 별도의 서버 설정 없이 웹 브라우저에서 직접 실행할 수 있습니다.

1.  프로젝트 파일을 로컬 컴퓨터에 다운로드합니다.
2.  다운로드한 폴더에서 `index.html` 파일을 찾습니다.
3.  `index.html` 파일을 웹 브라우저(Chrome, Firefox, Edge 등)로 드래그 앤 드롭하거나, 파일을 더블 클릭하여 엽니다.

## 💡 앞으로 개선 아이디어 (Future Improvement Ideas)

*   **태그 및 카테고리 (Tags & Categories):** 할 일에 태그나 카테고리를 추가하여 더욱 세분화된 관리를 가능하게 합니다.
*   **통계 및 진행률 (Statistics & Progress):** 완료된 할 일의 통계나 전체 진행률을 시각적으로 보여주는 기능을 추가합니다.
*   **반응형 디자인 (Responsive Design):** 모바일 기기에서도 최적화된 사용자 경험을 제공하도록 반응형 디자인을 적용합니다.
*   **마감 기한 설정 (Due Dates):** 각 할 일에 마감 기한을 설정하고, 마감 기한이 임박한 할 일에 대한 알림 기능을 추가합니다.
*   **사용자 인증 (User Authentication):** 여러 사용자가 각자의 할 일을 관리할 수 있도록 사용자 인증 기능을 도입합니다.
*   **애니메이션 및 UX 개선 (Animations & UX Improvements):** 사용자 인터페이스에 부드러운 애니메이션을 추가하고 전반적인 사용자 경험을 개선합니다.
